毕业论文格式要求

发表于 2026-04-05 分类于 杂项

小技巧

  • 打开新的修改工作和修改工作几乎做完时，ctrl + a 选中全部文字，设置字体 => 西文字体 => TimeNewRoma
  • 格式刷快捷键：Ctrl + Shift + C，Ctrl + Shift + V
  • 左上角文件 => 选项 => 视图 => 格式标记 => 勾选全部

基础格式

摘要

  • 摘要标题 ：为一级标题，中间两空格！
  • 摘要正文 ：格式为正文
  • 关键词 ：黑体，小四，无需首行缩进，关键词之间用 ；隔开

ABSTRACT（英文摘要）

  • 英文摘要标题 ：TimeNewRoma，三号，固定值20磅，居中，前后空一行，空行格式为正文
  • 英文摘要正文： 格式为正文
  • 关键字 ：加粗

目录

  • 目录标题： 为一级标题，中间两空格！
  • 目录内容：格式为正文，注意符号与英文格式为 TimeNewRoma

参考文献

  • 参考文献标题：一级标题
  • 参考文献内容：五号，悬挂缩进2字符！注意符号与英文格式为 TimeNewRoma

致谢

  • 致谢标题： 为一级标题，中间两空格！
  • 致谢内容：格式为正文，注意符号与英文格式为 TimeNewRoma

一级标题

1 绪 论

  • 黑体，三号，固定值20磅，居中，前后空一行，空行格式为正文

二级标题

1.1 研究目的及意义

  • 中文：黑体，小三，固定值20磅，大纲级别：2级
  • 其他字符：TimeNewRoma，设置方式见 小技巧

三级标题

​ 1.1.1 国内研究现状

  • 中文：黑体，四号，固定值20磅，大纲级别：3级
  • 其他字符：TimeNewRoma，同上

正文

  • 中文：宋体，小四！，固定值20磅，首行缩进
  • 其他字符：TimeNewRoma，同上

图片

  • 格式：嵌入式，单倍行距！（行距设置为固定值会压缩图片！），居中显示，无首行缩进！
  • 图注：位于图片下方，居中显示，五号，无首行缩进！
  • 图注内容：图x.x 图名，x.x代表当前x章的第x张图，如第二大章的第三张图：图2.3 xxxxxxxx

表格

  • 格式：三线表，上下边框为1.5磅实线，表头下边框为0.5磅实线
  • 内容格式：
    • 中文：五号，宋体，是否居中，根据具体情况判断，一般格式没有太乱的就不管
    • 其他字符：TimeNewRoma，其他同上
  • 表注：位于表格上方，居中显示，五号，无首行缩进！
  • 表注内容：表x.x 表名，x.x代表当前x章的第x张表，如第二大章的第三张表：表2.3 xxxxxxxx
  • 续表：当所有格式调整至差不多时，表格无法放在一页且没有办法较好调整至一页时，需要续表，在表格需要断掉的单元格下按 Ctrl + Shift + 回车，断开表格，在新的表格上方添加新的表注，格式为原表注最后 + （续）

代码

  • 格式：0.5磅实线边框全包裹，如出现边框不正常显示，则修改代码内容格式，不要有缩进
  • 内容格式：小四

其他格式

页眉

  • 下划线使用表格下边框
  • 格式：五号，固定值20磅，只有一行，无多余空行
  • 左侧为 重庆工程学院本科生毕业设计（论文），中间使用空格隔开，右侧为 域代码
  • 域代码插入方式：插入 => 文档部件 => 域 => 样式引用 => 样式名 => 一级标题（你设置的大纲级别为1的样式名），设置你要引用的文本格式为一级标题

注意点

  • 在调整页眉格式时，选中上方菜单栏，页眉同前节取消勾选，奇偶页不同取消勾选，同前节取消勾选

页脚

  • 摘要起，以罗马数字到目录
  • 绪论起，以阿拉伯数字到论文结束
  • 格式：居中，五号
  • 设置方式：点击页脚 => 页码设置，重新编号
