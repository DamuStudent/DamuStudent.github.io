一、常用方法

1.变量方法

  • toFixed 保留 x 位小数

2.数组方法

  • forEach 遍历数组，无返回值
  • map 遍历数组并返回一个新数组
  • filter 遍历数组并返回结果为true的元素

回调函数：element - 当前遍历的元素，index - （可选）当前元素索引，array - （可选）原始数组本身

返回值：true - 保留元素，false - 过滤元素

  • push(item) 在数组末尾添加元素，修改原数组，返回新长度
  • pop 删除数组末尾元素，修改原数组，返回被删除的元素
  • unshift 在数组开头添加元素，修改原数组，返回新长度
  • shift 删除数组开头元素，修改原数组，返回被删除的元素
  • splice(start,deleteCount,item1,item2,…) 从 start 索引开始删除 deleteCount 个元素，并插入新元素，修改原数组，返回被删的元素组成的数组
  • indexOf(item) 返回元素第一次出现的索引，未找到返回-1
  • includes(item) 检查是否包含某元素，返回布尔值
  • find(callback) 返回第一个满足条件的元素，未找到返回undefined
  • findIndex(callback) 返回第一个满足条件的元素的索引
